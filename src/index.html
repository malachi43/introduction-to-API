HOTEL MANAGEMENT API

This is a Hotel Management API. This API aids you to carry out the bare minimum of the basic services a Hotel Management API should offer.

    Available Endpoints:

  1. GET

      ROOM
      1. endpoint: https://introduction-to-api.onrender.com/api/v1/rooms - fetch all available rooms
      2. endpoint: https://introduction-to-api.onrender.com/api/v1/rooms/{roomId} - fetch single room

      ROOM TYPE
      1. endpoint: https://introduction-to-api.onrender.com/api/v1/room-types - fetch all room types.

  2. POST

      ROOM
      1. endpoint: https://introduction-to-api.onrender.com/api/v1/rooms - create a new room.

      ROOM TYPE
      1. endpoint: https://introduction-to-api.onrender.com/api/v1/room-types - create a new room type.

  3. DELETE

      ROOM
      1. endpoint: https://introduction-to-api.onrender.com/api/v1/rooms/{roomId} - delete a specific room with roomId.

  4. PATCH

      ROOM
      1. endpoint: https://introduction-to-api.onrender.com/api/v1/rooms/{roomId} - edit a specific room with roomId.
